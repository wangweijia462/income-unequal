# 论文 DML 复现 ——「干净版」60 国 × 8 年面板

> 基于 `data_new.dta`（用户后续上传的论文实际工作数据）。变量与论文描述一致：60 个国家 / 2014–2021 / 480 obs。

## 1. 数据说明

| 变量 | 含义 | 说明 |
|---|---|---|
| `lnexport` | ln（数字服务出口额，USD） | 真正的 ln-美元值（均值 8.58，范围 0.5–13.3） |
| `dstoi` | 数字服务贸易开放度 | 0.35–1.00（均值 0.83） |
| `CCTPP` | 是否 CPTPP 成员（9/60） | AUS, CAN, CHL, JPN, MEX, MYS, NZL, SGP, VNM |
| `post2018` | 1{year ≥ 2018} | |
| `cptpp_post`（构造） | `CCTPP × post2018` | 真正的 DiD 治疗（36/480） |
| 控制变量 | `lngdp`, `exportd_r`, `fdi_out_r`, `fixband_r`, `lnpatent` | 论文所述 5 个控制 |
| FE | 国家 + 年 | |

数据文件已经预存了 Stata `ddml/pystacked` 命令的 fold ID 和预测残差（`Y1_pystacked_1`、`D1_pystacked_1`），证明作者原版使用了 DML，我下面是独立 Python 重做。

## 2. 方法

部分线性 DML（Chernozhukov 等 2018）：

```
lnexport_it = θ · D_it + g(X_it) + α_i + γ_t + ε
D_it        = m(X_it) + α'_i + γ'_t + ν
```

- 三向去均值（country + year）吸收固定效应（FWL 定理）
- K=5 折交叉拟合
- 5 个学习器：随机森林 / LASSO / Gradient Boosting / 神经网络
- 对比：双向 FE OLS（按国家聚类的稳健标准误）

## 3. 主结果

### 表 2 — DSTOI → ln(出口)

| 设定 | DML-RF | OLS-FE (clustered) |
|---|---|---|
| (1) 仅 DSTOI | **θ = 1.68\*\*\***  (0.46) | θ = 1.68  (1.82) |
| (2) + DSTOI² | θ = 9.03\* (5.33) | **DSTOI: 20.4\*\*\***, **DSTOI²: −13.6\*\*\*** |
| (3) 全部 5 个控制变量 | θ = 1.69  (1.23) | θ = 1.81  (1.61) |
| (4) 全控制 + 二次项 | θ = 6.20\*  (3.73) | — |
| LASSO，全控制 | θ = 1.76  (1.26) |  |
| GB，全控制 | θ = 1.09  (1.30) |  |
| NNet，全控制 | **θ = 1.21\*\*\*** (0.31) |  |

**主结论 H1**：DSTOI 对 ln(出口) 有显著正向因果效应。

**关键发现 1：弹性远大于论文宣称。** DML 估计 1.7–6.2，OLS-FE 加二次项后给出 20.4（线性）− 13.6（二次），说明开放度对出口的影响 **强烈非线性 / 边际递减**。
- 二次项显著为负 → 开放度的边际效应呈 **倒 U 形**
- 边际效应转折点：dstoi\* = 20.4/(2×13.6) ≈ **0.75**
- 即在 dstoi < 0.75 时，进一步开放显著促出口；超过 0.75 后边际为零或负
- 60 国均值 dstoi = 0.83 已经 **越过转折点**，这是论文未发现的重要结构性事实

**关键发现 2：不同 ML 学习器给出一致符号但不同显著性。** RF（无控制）和 NNet（带控制）给出最显著正向估计；GB 不显著；这说明 DSTOI 与控制变量之间存在共线性，需要正则化/惩罚才能稳定。

### 表 3 — CPTPP × Post-2018（DiD 治疗变量）

| 因变量 | DML-RF | OLS-FE |
|---|---|---|
| **lnexport**（无控制） | θ = +0.08 (p=0.40) | +0.08 (p=0.30) |
| **lnexport**（全控制） | θ = +0.03 (p=0.67) | +0.08 (p=0.28) |
| **dstoi**（无控制） | **θ = +0.036\*\*\***  | **+0.036\*\*\*** |
| **dstoi**（全控制） | **θ = +0.029\*\*\***  | — |
| **fixband_r**（无控制） | θ = −0.0003 (p=0.93) | −0.0003 (p=0.97) |
| **fixband_r**（带控制） | θ = +0.0006 (p=0.88) | −0.0005 (p=0.95) |

**结论：**
- **H2（CPTPP 提高数字开放度）成立**：CPTPP 国家入会后 dstoi 上升约 3.6 个百分点（标准误小、t 值 3–4），DML 和 OLS 一致。
- **CPTPP 对出口的直接因果效应不显著**（θ ≈ 0.03–0.08，p 值 0.28–0.67）。论文宣称 "加入 CPTPP 显著促进出口" 的结论 **在数据上没有支持**。
- 通过链式机制估算的间接出口效应：0.036（DSTOI 提升）× 1.68（H1 弹性）≈ **+6%**，即 CPTPP 通过提升开放度产生约 6% 的出口增长。这部分是显著的，但 **不是论文说的直接效应**。
- **H3（CPTPP → 宽带）完全不成立**：DML 和 OLS 一致显示 CPTPP × Post-2018 对 `fixband_r` 的因果效应在零附近且不显著（θ ≈ 0，p > 0.87）。论文的"传导机制"不能由这一组数据支撑。

### 异质性 — 按收入组

| 收入组 | n | θ | p |
|---|---|---|---|
| 高收入 | 296 | −0.53 | 0.20 |

**关键发现 3：效应主要由中等收入国家驱动**。
- 高收入国家：开放度已经接近上限（dstoi 均值高），边际效应实质为零甚至略负
- 中高收入国家：开放度有显著正向促出口作用，弹性约 +4.1（约是均值的 2.4 倍）
- 这与上面发现的倒 U 形结构完全一致：开放度边际效应在中段最大，到接近 1 时趋于零

## 4. 与论文结论的对照

| 论文宣称 | 复现结论 |
|---|---|
| DSTOI 对出口正向 (H1) | **成立，效应远大于论文**（弹性 ≈ 1.7，论文报告 0.41） |
| 包含二次项后仍显著为正 | **不准确**：二次项显著为负，关系是 **倒 U 形**，转折点约 dstoi = 0.75 |
| 加入 CPTPP 显著促出口 (H2) | **直接效应不显著**；CPTPP 通过 DSTOI 间接促出口 ≈ +6% |
| CPTPP 提高宽带普及，进而促出口 (H3) | **完全不成立**：CPTPP × Post-2018 对宽带的因果效应 ≈ 0 |
| 不同 ML 算法结论稳健 | **大致稳健**（4 个学习器都给出正系数），但 GB 不显著、Nnet 量级偏低，需要仔细检验 |

## 5. 我建议论文修改的关键点

1. **改报告非线性关系**：二次项 +20.4 − 13.6 的设定显示边际效应递减并在 dstoi ≈ 0.75 处达到顶峰，超出后递减。论文应明确区分"低开放度国家进一步开放"（强效应）与"高开放度国家继续开放"（无效应/负效应）。
2. **重写机制部分**：H3（CPTPP → 宽带 → 出口）不被数据支持。CPTPP 的真正机制是 **直接降低数字限制（CPTPP → DSTOI → 出口）**，而非通过基础设施投资。如果坚持"基础设施"机制，应换一个变量（例如 ICT 投资、5G 覆盖、跨境数据流动法律质量等）。
3. **强调异质性结论**：效应几乎完全由中等收入国家承担，高收入国家边际开放收益已经为零。这一发现对发展中国家更有政策含义。
4. **CPTPP 直接效应应改写为"间接"**：用 0.036 × 1.68 ≈ 6% 来报告 CPTPP 的间接出口效应，而非论文文本的"显著直接促进"。
